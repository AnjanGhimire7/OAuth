
## clustering application using pm2

pm2 Install :
yarn global add pm2
run :
pm2 start src/index.js -i max //this will provide number of cluster as much as cpu core availabe on the computer.


## Technologies Used

- Node.js
- Express.js
- MongoDB
- JSON Web Tokens (JWT) for authentication
- passport js
- google oauth
